# Turboplay

Offline arcade with short games for mobile, tablet, and laptop.

## Includes

- `Cabin 2048`
- `Aisle Lights`
- `Radar 30s`
- offline caching with a `service worker`
- PWA-style installation

## Local development

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.
