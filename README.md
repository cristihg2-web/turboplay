# Turboplay

Arcade offline con minijuegos cortos para movil, tablet y portatil.

## Incluye

- `Cabina 2048`
- `Luces de pasillo`
- `Radar 30s`
- cache offline con `service worker`
- instalacion tipo PWA

## Desarrollo local

```bash
python3 -m http.server 4173
```

Luego abre `http://localhost:4173`.
